Introduccion a Git

  • git init: Inicializar repositorio
  • git status: Verifica el estado de mis archivos
  • git add file_name: guarda el archivo en el area de preparacion
  • git commit -m "message": Agrega una version de nuestro aplicacion
  • git log: muestra las versiones de mi aplicacion

branching

  • git branch: Muestra las ramas de mi aplicacion
  • git branch new_branch_name: Crea una nueva rama de la aplicacion
  • git checkout branch_name:Permite el moverse entre ramas
  • git branch_name -d: Elimnar una branch
